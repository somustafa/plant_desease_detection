The primary objective of this project is to develop a real-time plant disease detection system integrated into a mobile robot. By utilizing the lightweight YOLOv8-Nano architecture, the system is specifically optimized to provide high-speed, low-latency inference on resource-constrained hardware like the Raspberry Pi. This setup allows the robot to autonomously identify and localize diseases in lettuce, strawberry, and tomato leaves while in motion. The ultimate goal is to enable precision agriculture through efficient, AI-driven edge diagnostics.
